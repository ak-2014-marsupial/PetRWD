# Фіча FontSizePicker: Керування Розміром Шрифту

Фіча `FontSizePicker` є частиною віджета `ThemeSettingsEditor` і дозволяє користувачам керувати глобальним розміром шрифту всього додатку, що покращує доступність (Accessibility).

## Принцип Роботи

1.  **Вибір користувача:** Компонент `FontSizePicker` рендерить набір опцій (наприклад, 'S', 'M', 'L'), які користувач може обрати.

2.  **Зміна класу:** При виборі, JavaScript-логіка встановлює на кореневий елемент `<html>` відповідний клас, наприклад, `font-size-m`.

3.  **Масштабування через CSS:**
    - Для кожного класу (`font-size-s`, `font-size-m`, `font-size-l`) в CSS визначено різне значення для базової CSS-змінної `--font-size-base`.
    - Усі розміри тексту в додатку (для `p`, `h1`, `h2` і т.д.) визначені в відносних одиницях `rem`. Оскільки `rem` розраховується відносно розміру шрифту кореневого елемента (`<html>`), зміна `--font-size-base` призводить до пропорційного та узгодженого масштабування всього тексту на сайті.

Такий підхід є надзвичайно ефективним для реалізації налаштувань доступності та персоналізації.
